This game was made from Jonas Schmedtmann's Complete Javascript course on Udemy.
GAME RULES:

  1. The game has 2 players, playing in rounds
  2. In each turn, a player rolls a dice as many times as he whishes. Each result get added to his score
  3. The score limit can be set to any positive number
  4. If the player rolls a 1, all his ROUND score gets lost. After that, it's the next player's turn
  5. If a a player rolls two 6's in a row then their total score gets reset to 0
  6. The player can choose to 'Hold', which means that his ROUND score gets added to his total score. After that, it's the next player's turn
  7. The first player to reach 100 or set target on the total wins the game

Changes made to the original are:

  1. Animation when a 1 or two 6's are rolled
  2. Showed losing die for a second before disappearing
  3. Disabled 'Roll Dice' whilst animation is playing
  4. Set score is not applied until button is pressed
